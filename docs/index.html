Powerful Features

Everything you need to give your AI a persistent, organized memory.

Intent-Based Storage

AI expresses what to store, Palace determines where. Automatic organization based on your vault structure.

Multi-Vault Support

Work with multiple Obsidian vaults. Set read/write permissions per vault. Keep work and personal knowledge separate.

Auto-Linking

Automatically creates [[wiki-links]] between related notes. Your knowledge graph grows organically.

Full-Text Search + Dataview

Fast FTS5 search with BM25 ranking. Execute Dataview queries (DQL) directly from AI.

Atomic Notes

Large content automatically splits into hub + child notes. Keeps your vault clean and navigable.

Works With Any MCP Client

Claude Desktop, ChatGPT, or any MCP-compatible AI. Not locked to any single provider.

How It Works

Get started in minutes with four simple steps.

1

Install via npm

One command to install globally:

npm install -g obsidian-palace-mcp
2

Configure Your Vault

Point Palace to your Obsidian vault. Set up via environment variable or config file. Supports multiple vaults with different permissions.

3

Connect to Your AI

Add Palace as an MCP server in Claude Desktop, ChatGPT, or any MCP-compatible client. The AI gains access to 20+ knowledge management tools.

4

AI Remembers Everything

Your AI can now store research, commands, decisions, and learnings. It retrieves context from previous sessions. Knowledge compounds over time.
